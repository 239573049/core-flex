# CoreFlex

## 框架描述

CoreFlex是一个支持`.NET 6`,`.NET 7`,`.NET 8`的快速开发框架，也提供`MasaFramework`相关框架的集成提供更多功能模块,
集成了一些常用的功能模块，可以快速开发企业级应用,提供`Blazor`的一些JS操作工具库。


## 实现目标

- [x]  JS工具操作模块
- [ ]  Serilog日志模块
- [ ]  限流中间件模块
- [ ]  审计日志模块
- [ ]  虚拟文件模块(LiteDb实现和本地文件实现)
- [ ]  系统设置模块
- [ ]  后台任务模块
- [ ]  提供`ChatGpt`客户端案例模块
